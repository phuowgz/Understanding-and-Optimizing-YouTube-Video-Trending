Developed a comprehensive data-driven framework to analyze and predict YouTube video trending potential, focusing on factors like view velocity, engagement metrics, and temporal features across five countries (US, UK, Brazil, Canada, Mexico) link Kaggle https://www.kaggle.com/datasets/rsrishav/youtube-trending-video-dataset?select=CA_youtube_trending_data.csv.

Preprocessed a Kaggle dataset of over 200,000 videos, handling missing values, outliers, and feature engineering to prepare data for modeling.

Tools & Technologies: Python, XGBoost, LightGBM, SHAP, TextBlob, spaCy; dataset sourced from Kaggle, Github.
